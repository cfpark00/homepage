## Key Changes from Previous Structure

### Route Organization
- Moved all main pages to `(default)/` route group for shared layout
- Isolated `/simple` page with its own minimal layout
- Removed `(main)/` route group (consolidated into `(default)/`)

### Asset Organization
- Created `/public/images/` directory for all images
- Moved profile picture and orchestra logo to proper locations
- CV PDF accessible at root public level

### Component Updates
- `chat-box.tsx` now supports dual mode (Virtual/Real Core)
- Added `sonner.tsx` for toast notifications
- Navigation order: Home, CV, Ongoing, Publications, Blog, News (Research hidden)

### Content Structure
- Blog posts remain in `/content/blog/`
- Publications data in `/lib/publications-data.ts`
- CV data embedded in component

### Page Features
- Homepage: Two-column hero with profile image right, flying publications animation (50% transparent sections)
- CV: Rectangular profile image left of header
- Ongoing: Square thumbnail cards with thin text
- Simple: True standalone 1990s HTML page with rectangular profile image floating left
- ChatBox and Newsletter components commented out (not deployment ready)

## Navigation Flow
- Persistent top bar on all pages except /simple
- Email copies to clipboard with toast
- /simple has no navigation, just back link
- Footer minimal padding (py-3/4)

## Styling System
- Tailwind CSS with custom theme
- CSS variables for theming
- Dark mode support throughout
- Special raw HTML styling for /simple page
- Toast notifications bottom-right

## Deployment Configuration
- **Git Repository**: git@github.com:cfpark00/homepage.git
- **Vercel Project**: "homepage" (not "mysite")
- **Vercel Settings** (for monorepo):
  - Root Directory: `apps/web`
  - Framework Preset: Next.js
  - Build Command: `pnpm build`
  - Output Directory: `.next`
  - Install Command: `pnpm install`
